# 🍽️ buffet-cdn

Репозиторий со статическими изображениями продуктов университетского буфета, подготовленными для использования в мобильном приложении.

---

## 📦 Содержимое

- 43 изображения продуктов
- Исходники скачаны с официального сайта университета
- Выполнен AI-upscale для повышения качества
- Конвертация и оптимизация в формат **JPEG**
- Подготовлены для раздачи через **jsDelivr CDN**

---

## 🎯 Назначение

Репозиторий используется как статическое CDN-хранилище изображений.

Подключение через jsDelivr позволяет:

- снизить нагрузку на backend
- ускорить загрузку изображений
- обеспечить стабильную раздачу через глобальный CDN
- упростить архитектуру приложения (JAMStack-подход)

## 🛠 Pipeline обработки

1. Загрузка изображений с сайта университета  
2. AI-upscale  
3. Сжатие и конвертация в JPEG  
4. Загрузка в GitHub  
5. Раздача через jsDelivr  

---

## 📌 Примечание

Репозиторий используется исключительно как хранилище изображений и не содержит backend-логики.
